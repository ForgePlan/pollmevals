# Calibration samples

Calibration samples are controlled submissions with expected scores. They are used to verify that judges and tasks discriminate between excellent, good, mediocre, poor and broken outputs.

Each active task should eventually have:

- perfect sample: expected 9-10
- good sample: expected 7-8
- mediocre sample: expected 5-6
- poor sample: expected 2-3
- broken sample: expected 0-1
