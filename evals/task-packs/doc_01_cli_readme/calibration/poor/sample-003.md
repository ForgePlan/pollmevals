# pollmevals fetch-task

## Overview

CLI for POLLMEVALS task packs.

## Quick start

```sh
pollmevals fetch-task ls
pollmevals fetch-task pull be_01
pollmevals fetch-task run be_01
```

## Commands

### ls

Lists packs. Same as `list`.

### pull

Downloads a pack to the local cache.

### show

Shows a pack.

### validate

Validates a pack.

### run

Runs a pack.

### clean

Clears the local cache.
